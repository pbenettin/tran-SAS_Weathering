# Automatic README file generated for 'SyntheticData'.csv

The file 'SyntheticData'.csv includes tabular data at hourly timesteps:

- date, formatted as datetime dd-mmm-yyyy HH:MM:SS (e.g. 01-Jan-2020 00:00:00)
- J, precipitation in mm/h
- Q, streamflow in mm/h
- ET, evapotranspiration in mm/h
- Cin, input tracer concentration (here, d18O in permil)
- wi, indicator of wetness index, non-dimensional and confined within [0-1]. Here, it is the ratio between soil runoff versus total runoff
- measC_Q, output sampled concentration (here it's all NaN)

Missing values are indicates as NaN
